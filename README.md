Códigos sobre regressão linear desenvolvidos no segundo semestre de 2025. Resolução das listas inciais da disciplina
TP555 do mestrado em telecomunicações do Inatel. Os exercícios abordam de forma aprofundada técnicas de regressão
são implementados algoritmos de gradiente descendente (em batelada, estocástico e mini-batches). Também são estudadas
técnicas de otimização de hiperparâmetros, como o grid search e demais técnicas como adaptive learning rate e decay.
Esses códigos tem como objetivo fundamental o entendimento e implementação matemática dos algoritmos considerados a 
base dos demais tópicos de machine learning, permitindo uma evolução sólida e maior entendimento de técnicas mais avançadas.
