# TP555

Este repositório reúne códigos e exercícios sobre **regressão**, desenvolvidos no segundo semestre de 2025 como parte das listas iniciais da disciplina **TP555**, do Mestrado em Telecomunicações do Instituto Nacional de Telecomunicações (Inatel).

Os exercícios exploram, de forma aprofundada, os fundamentos matemáticos e computacionais das técnicas de regressão, com ênfase na implementação dos algoritmos a partir de seus princípios teóricos.

## Tópicos abordados

* Análise e visualização de superfícies de erro;
* Regressão linear e análise de modelos preditivos;
* Implementação de **gradiente descendente em batelada (Batch Gradient Descent)**;
* Implementação de **gradiente descendente estocástico (Stochastic Gradient Descent — SGD)**;
* Implementação de **gradiente descendente por mini-batches (Mini-Batch Gradient Descent)**;
* Otimização de hiperparâmetros por **Grid Search**;
* Estratégias de **Adaptive Learning Rate** (*Adagrad*, *RMSProp* e *Adam*);
* Técnicas de **Learning Rate Decay**.

## Objetivos

O principal objetivo deste material é promover o entendimento conceitual e a implementação matemática dos algoritmos que constituem a base do aprendizado de máquina moderno. O domínio desses fundamentos proporciona uma evolução mais sólida e um maior embasamento para o estudo e desenvolvimento de técnicas mais avançadas de **Machine Learning**.
